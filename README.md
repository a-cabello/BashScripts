<h2><b>hostScan</h2></b>
<b>How do I try it?:</b>

(1) Clone the repository:
```bash
git clone https://github.com/a-cabello/BashScripts.git
```
(2) Move to the directory
```bash
cd BashScripts
```
(3) Run the script
```bash
./hostScan.sh
```
